Little Wise Kids — Thunderbird email signatures

Files:
- 00-template-edit-me.html
- Four sample staff signatures

Before use:
1. Upload the final logo to:
   https://littlewisekids.co.uk/assets/images/logo.png
2. Open each HTML file in a text editor.
3. Replace the sample name, title and email address where required.
4. Keep the rest of the HTML unchanged.

Thunderbird:
1. Open Thunderbird.
2. Open Account Settings.
3. Select the relevant email account.
4. Under Signature text, enable “Attach the signature from a file instead”.
5. Choose the matching .html file.
6. Ensure the account composes messages as HTML.
7. Send a test message to Gmail and Outlook and check desktop and mobile rendering.

Each Thunderbird account must point to its own signature file.
